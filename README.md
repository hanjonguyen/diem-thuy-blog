# Coach Diễm Thúy — Blog giới thiệu

Trang blog tự giới thiệu của **Coach Diễm Thúy** — Dược sĩ, coach xây kênh thương hiệu cá nhân cho bác sĩ/dược sĩ, Giám đốc cộng đồng BNI.

> "Chiếc xe lăn không phải dấu chấm hết. Nó là một đòn bẩy."

## Nội dung
Một trang HTML tĩnh tự chạy (`index.html`), giao diện liquid-glass tông xanh noti.vn (`#0061ff → #00c2ff`), font Inter + Playfair Display, hỗ trợ sáng/tối.

## Thêm ảnh thật
Đặt ảnh vào thư mục `img/` với tên `diem-thuy-hero.jpg` để hiện ảnh chân dung ở hero (thay cho monogram mặc định).

## Triển khai
- Static site — deploy thẳng lên Vercel/Netlify/GitHub Pages, không cần build.
